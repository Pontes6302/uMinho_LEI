## 🚀 Como utilizar este repositório

Se quiseres explorar algum projeto localmente:

1. **Clone o repositório:**
   ```bash
   git clone --recursive https://github.com
   ```
   *(Nota: O uso do `--recursive` é importante devido aos submodules como o de LI3).*

2. **Navegue até a pasta da UC desejada:**
   ```bash
   cd "2 Ano/1 Semestre/LI3"
   ```
